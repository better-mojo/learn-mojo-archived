# Mojo 101:

- Mojo 新手教程

## 初始化开发环境：

- ✅️ python3.10
- ✅️ poetry
- ✅️ jupyter notebook

```bash


task install


```

- 设置环境变量:

```ruby

echo 'export MODULAR_HOME="/Users/dev/.modular"' >> ~/.zshrc
echo 'export PATH="/Users/dev/.modular/pkg/packages.modular.com_mojo/bin:$PATH"' >> ~/.zshrc
source ~/.zshrc

```
